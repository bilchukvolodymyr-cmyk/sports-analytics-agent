# Sports Analytics Agent 🏆

Агент для збору та аналізу статистики спортивних подій. Отримує детальну аналітику за командами за останні 6 місяців.

## Можливості

- 📊 Аналіз матчів за 6 місяців
- 🟨 Статистика жовтих карток
- ⚽ Кількість кутових
- 📈 Результати матчів
- 📉 Графіки та таблиці для швидкого розуміння
- 🌍 Підтримка коман�� з різних країн

## Встановлення

```bash
git clone https://github.com/bilchukvolodymyr-cmyk/sports-analytics-agent.git
cd sports-analytics-agent
pip install -r requirements.txt
```

## Використання

```bash
python main.py "Аргентина" "Єгипет"
```

## Вивід

- Таблиця з матчами та статистикою
- Графіки порівняння команд
- Детальна аналітика за 6 місяців

## Джерела даних

- football-data.org (безплатний API)
- FIFA/UEFA офіційні дані
- ESPN (публічні дані)
